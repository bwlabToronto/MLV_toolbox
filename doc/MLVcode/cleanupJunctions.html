cleanupJunctions

PURPOSE

cleanedJunctions = cleanupJunctions(Junctions, Thresh)

SYNOPSIS

function cleanedJunctions = cleanupJunctions(Junctions, Thresh)

DESCRIPTION 

 cleanedJunctions = cleanupJunctions(Junctions, Thresh)
       cleans up junctions by merging junctions that are within Thresh pixels of
       each other.
